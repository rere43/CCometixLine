# @kei233/ccline

CCometixLine - High-performance Claude Code StatusLine tool

## Installation

```bash
npm install -g @kei233/ccline
```

If you have installed the official package before (or any other package that provides the `ccline` command), force reinstall to ensure `ccline` is replaced:

```bash
npm i -g --force @kei233/ccline@latest
```

## Usage

After installation, ccline is automatically configured for Claude Code at `~/.claude/ccline/ccline`.

You can also use it directly:

```bash
ccline --help
ccline --version
```

## For Users in China

Use npm mirror for faster installation:

```bash
npm install -g @kei233/ccline --registry https://registry.npmmirror.com
```
